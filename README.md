# Ship Shape style frames

Three interactive art-direction studies for the software-engineering deckbuilder vertical slice.

## Review locally

Serve this directory with any static file server, for example:

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

Use the tabs to compare Live Canvas, Launch Broadcast, and Code Cartoon. The content remains the same across all three directions. Cards and roster portraits are lightly interactive, and the direction tabs support arrow-key navigation.
